1 + 1 = 2: Formal Foundations and Meaning

The statement 1 + 1 = 2 is elementary arithmetic, yet its truth depends on how “1”, “+”, and “2” are formally defined.

Peano Arithmetic
In Peano arithmetic, natural numbers start with 0 and a successor function 
𝑆
S. We define:

1=𝑆(0)
1=S(0)
2=𝑆(1)
2=S(1)

Addition is defined recursively:

𝑎+0=𝑎
a+0=a
𝑎+𝑆(𝑏)=𝑆(𝑎+𝑏)
a+S(b)=S(a+b)

Using these axioms:

1 + 1 = S(0) + S(0)
   = S(S(0) + 0)
   = S(S(0))
   = 2

Alternatively, using 
𝑎+1=𝑆(𝑎)
a+1=S(a), we also get:

1 + 1 = S(1) = 2

Set-Theoretic Construction
In set theory (e.g., Von Neumann ordinals):

0 = ∅

1 = {0} = {∅}

2 = {0, 1} = {∅, {∅}}

and so on.

Cardinal addition is defined using disjoint unions:

If A and B are disjoint sets,
then 
∣𝐴∣+∣𝐵∣=∣𝐴∪𝐵∣
∣A∣+∣B∣=∣A∪B∣

Example:
A = {a}, B = {b} (disjoint singletons)
Then 
∣𝐴∣=∣𝐵∣=1
∣A∣=∣B∣=1,
and 
∣𝐴∪𝐵∣=2
∣A∪B∣=2,
so 
1+1=2

Formal Logic (Principia Mathematica)
In Principia Mathematica by Russell and Whitehead, arithmetic is derived from logic. 1 is defined as the class of one-element sets, and 2 as the class of two-element sets. Addition of cardinals is defined by union of disjoint classes.

After hundreds of pages, they state in proposition 54.43:
"From this proposition it will follow, when arithmetical addition has been defined, that 1 + 1 = 2."

Thus, 1 + 1 = 2 is a theorem of their logical system.

Philosophical Perspectives
Logicism (Frege, Russell): Arithmetic is derivable from logic. 1 + 1 = 2 is analytically true.

Kant (Transcendental Idealism): Arithmetic truths are synthetic a priori—not just logical tautologies.

Formalism (Hilbert): Math is symbol manipulation. 1 + 1 = 2 is true within a formal system like Peano arithmetic.

Intuitionism: 1 + 1 = 2 is true because we can construct 2 from 1.

Wittgenstein (Tautology View): 1 + 1 = 2 is a tautology—two ways of saying the same thing.

Empiricism vs Platonism: Empiricists saw math as generalizing experience; Platonists say 1 + 1 = 2 is a timeless truth.

Conclusion
Each view explains the truth of 1 + 1 = 2 in different terms—by definition, by construction, by proof, or by language. It remains a core example in logic and philosophy about the meaning of mathematical truth.
